# Money Sprint 16 — Draw Tracker Sort + Search

## Goal
Make the French Open draw tracker more useful for real users after the active-player API fix.

## Changes
- Active players are now sorted by next match urgency instead of alphabetically.
- Live players appear first.
- Upcoming players are ordered by match date/time.
- The selector includes a search input for player or next opponent.
- Player cards now show the next opponent where available.
- Tracker summary counters now show live, upcoming and hidden eliminated counts.

## Why this helps
Users landing from Google can quickly find the player they care about and understand whether the match is live, upcoming, or already filtered out. This improves retention and routes more users toward schedule, results and where-to-watch money pages.
