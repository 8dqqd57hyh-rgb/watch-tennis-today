# Money Sprint 10 — Country Broadcaster Hub

## Goal
Improve monetization-oriented navigation without adding popups, sticky banners, or aggressive ads.

## What changed
- Added a new `/watch-tennis-in` hub page that links to all country broadcaster guides.
- Added structured data for the country guide collection.
- Added calm internal links to schedule, tomorrow schedule, compare, broadcast finder, and streaming services pages.
- Added one optional inline email signup block at the bottom of the page only.
- Added `/wimbledon-live-stream` as a safe alias to prevent 404s from older/internal Wimbledon live links.
- Updated the homepage Wimbledon spotlight link to use the existing `/wimbledon-live` route.
- Updated sitemap entries for the new country hub and Wimbledon alias.

## UX rules followed
- No popups.
- No sticky bars.
- No forced signup.
- No fake stream claims.
- Clear disclaimer that the site does not host streams.
